# TEMPERATURE-PREDICTION
This project predicts temperature using machine learning techniques. It is built using historical weather data and helps in forecasting future temperature based on features like humidity, wind speed, and pressure.

# Features
Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Machine learning model training
Temperature prediction
Data visualization

# Technologies Used
Python
Pandas
NumPy
Matplotlib
Scikit-learn

# Results
The model predicts temperature with good accuracy and performs well on test data.
